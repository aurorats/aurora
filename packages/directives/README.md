# Aurora Directives

Aurora directive had the built-in directives for aurora project.

## `Install`

``` bash
npm i --save @ibyar/directives
```

``` bash
yarn add @ibyar/directives
```

# How to use:

```html
<div *if="x > 50" > x: is {{x}} </div>
```

## Supported Directive

- [x] *if
- [x] *for
- [x] *while
- [ ] *switch
- [ ] *class
- [ ] *style
